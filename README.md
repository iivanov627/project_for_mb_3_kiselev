# Пчелы и Медведь (Bee and Bear)
Киселев Группа

Краткое описание: Многопоточная Java-программа, реализующая классическую задачу синхронизации потоков "Пчелы и Медведь". Проект демонстрирует работу с потоками (Thread), синхронизацией (synchronized, wait/notify) и включает функционал статистики сбора и потребления меда. Проект предназначен для изучения многопоточного программирования и синхронизации в Java, а также для демонстрации работы классических паттернов синхронизации потоков.

## Стек
- **Язык:** Java 24
- **Сборка:** Maven
- **Ключевые библиотеки:** Стандартная библиотека Java (java.util.Random, java.lang.Thread)
- **Сервисы и интеграции:** Не требуются

## Быстрый старт

### Требования
- Java Development Kit (JDK) 24 или выше
- Apache Maven 3.6.0 или выше

### Установка и запуск

1. Клонируйте репозиторий или перейдите в директорию проекта:
```bash
cd "Киселев"
```

2. Скомпилируйте проект с помощью Maven:
```bash
mvn compile
```

3. Запустите программу:
```bash
mvn exec:java -Dexec.mainClass="org.example.BeeAndBear"
```

Или альтернативный способ:
```bash
# Компиляция
mvn compile

# Запуск скомпилированного класса
java -cp target/classes org.example.BeeAndBear
```
